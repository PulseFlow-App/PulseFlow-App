# Pulse Roadmap

This roadmap maps the Pulse vision into two clear execution phases: **MVP 1** (core signal + usefulness) and **MVP 2** (adaptation + depth).

---

## MVP 1  - Core Signal & Daily Use

**Goal:**  
Deliver a simple, valuable daily experience that turns raw inputs into a clear signal users want to check every day.

### Core Features

- **Daily Pulse Score (0â€“100)**
  - Computed from user-submitted inputs
  - Updates in real time as data is added
- **Manual Data Input**
  - Meals (text or photo)
  - Movement / workouts
  - Sleep duration & quality
  - Mood / energy check-in
- **Basic AI Interpretation**
  - "What this means today"
  - Short, actionable suggestions
- **Daily vs Yesterday Comparison**
  - Lightweight trend awareness without overload
- **Minimal Dashboard**
  - Today's Pulse
  - Recent inputs
  - One primary recommendation

### AI Capabilities (MVP 1)

- Single-day reasoning
- Basic pattern detection
- Ingredient-based meal suggestions from fridge photos
- Simple recovery & activity feedback

### Token Utility (MVP 1)

- **$PULSE** exists on [Pump.fun](https://pump.fun/coin/5ymQv4PBZDgECa4un7tYwXSVSWgFbfz79qg83dpppump)  - no token contract to create
- Token-gated **Premium Tier**
- Premium unlocks:
  - Higher daily request limits
  - No per-request fees (excluding gas)
  - Access to Pulse Score explanations

### What MVP 1 Proves

- Users understand and trust the Pulse score
- People return daily
- AI insights feel helpful, not overwhelming

---

## MVP 2  - Adaptation & Personal Baselines

**Goal:**  
Transform Pulse from a daily snapshot into a personalized system that learns the user over time.

### Expanded Features

- **Personal Baseline Modeling**
  - User-specific norms replace global averages
  - Pulse score adapts to the individual
- **Long-Term Trend Tracking**
  - Daily vs weekly vs historical views
  - Contextual trend summaries
- **Multi-Input Reasoning**
  - Combined analysis across food, movement, sleep, and body data
- **Advanced Recommendations**
  - Context-aware suggestions
  - Behavior-aware guidance (e.g. recovery vs performance days)

### Nutrition & Lifestyle (MVP 2)

- Weekly adaptive meal suggestions
- Personal nutrition pattern detection
- Ingredient-based planning using user history
- Energy and mood correlations

### Fitness & Body Data (MVP 2)

- Fitness progress interpretation
- Recovery vs strain insights
- Smart scale data analysis (unlimited)
- Body metric trend explanations (non-medical)

### Token Utility (MVP 2)

- Full premium feature access via locking (lock for as long as you want the paid tier)
- Unlimited AI requests
- Deep historical analysis
- Priority model updates and new insight modules

### What MVP 2 Unlocks

- Long-term user trust
- Clear personalization advantage over generic health apps
- Strong incentive to hold and lock Pulse tokens (lock for as long as you want the paid tier)

---

## Summary

- **MVP 1**: "Pulse helps me understand today."
- **MVP 2**: "Pulse understands me."

Both stages focus on everyday usefulness first, depth second  - ensuring Pulse grows with its users rather than overwhelming them from day one.
